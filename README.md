# BeforeMidnight

> Focus on what matters today.

BeforeMidnight is a minimalist daily reset task app designed to help you focus only on what matters today.

Unlike traditional productivity apps that create endless backlogs, BeforeMidnight automatically clears your tasks at midnight — giving you a fresh start every single day.

The app is intentionally simple, calm, and focused on daily clarity rather than long-term task accumulation.

---

## ✨ Features

- ⏳ Real-time countdown to midnight
- 📝 Daily task tracking with multi-line input support
- 🌙 Automatic reset every new day
- 📅 Yesterday review with unfinished task carry-forward
- 💾 Local storage persistence
- 🔒 Stored entirely on your device
- ⚡ Quick multi-task input (newline or comma separated)
- ⌨️ Keyboard shortcut support (`Ctrl/Cmd + Enter` to save)
- 📱 Fully responsive mobile design with smart autofocus
- ♿ Accessibility improvements with ARIA labels
- 🎯 Dynamic progress messaging
- ↩️ Undo delete for accidental task removal (5-second window)
- ✨ Smooth animations and transitions
- 🎨 Beautiful minimalist UI with subtle gradients
- 🚀 Instant task feedback and visual states

---

## 🧠 Philosophy

BeforeMidnight is built around one idea:

> Not everything needs to carry into tomorrow.

The app encourages:

- focus
- intentional action
- psychological closure
- clean daily resets

No accounts.
No cloud sync.
No endless productivity systems.

Just today.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/AnoopPushpad/BeforeMidnight.git
cd BeforeMidnight
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Lint the code:

```bash
npm run lint
```

---

## 📖 How to Use

1. **Add tasks** — Type your tasks in the text area, separated by newlines or commas
2. **Save** — Click "Save Today" or press `Ctrl/Cmd + Enter` (desktop)
3. **Complete** — Check off tasks as you finish them
4. **Tomorrow** — Your tasks reset at midnight; unfinished tasks can be carried forward
5. **Undo** — Accidentally deleted a task? Click "Undo" within 5 seconds
